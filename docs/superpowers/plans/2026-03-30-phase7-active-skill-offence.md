# Phase 7: Active Skill Resolution + Offence Calculations — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `active_skill.rs` and `offence.rs` so that the engine produces correct DPS, damage-type breakdown, crit, hit chance, ailment damage, DoT, and all other offence outputs for the five oracle builds: `melee_str`, `crit_spellcaster`, `ignite_dot`, `bleed_dot`, and `poison_dot`. At the end, `DATA_DIR=data cargo test -p pob-calc oracle` passes all five of these oracle tests within 0.1% tolerance.

**Architecture:** `active_skill.rs` resolves the main skill from the build's skill sets, sets `skillCfg` flags and conditions, and populates `env.player.main_skill` with a resolved `ActiveSkill`. `offence.rs` then consumes `env.player.main_skill` to run the full damage pipeline: base damage per type, conversion chain, inc/more modifiers, crit, hit chance, average hit, ailment damage, DoT, and total DPS outputs.

The key insight: offence calculations in POB operate on a `skillModList` — a `ModDb` that combines the player's global mods filtered through the active skill's flags (Attack/Spell/Melee/etc.) plus the skill's own granted-effect mods. In the Rust port, `ActiveSkill.skill_mod_db` is this filtered view.

**Tech Stack:** Rust 1.82, `pob-calc` (Phases 3–6 complete)

**Prerequisites:** Phase 6 complete. Oracle builds generated for `melee_str`, `crit_spellcaster`, `ignite_dot`, `bleed_dot`, `poison_dot` (see Task 1).

```rust
/// Used to determine how a skill interacts with the mod system.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct SkillTypeFlags(pub u64);

impl SkillTypeFlags {
    pub const NONE: Self = SkillTypeFlags(0);
    pub const ATTACK: Self = SkillTypeFlags(1 << 0);
    pub const SPELL: Self = SkillTypeFlags(1 << 1);
    pub const PROJECTILE: Self = SkillTypeFlags(1 << 2);
    pub const AREA: Self = SkillTypeFlags(1 << 3);
    pub const DURATION: Self = SkillTypeFlags(1 << 4);
    pub const MELEE: Self = SkillTypeFlags(1 << 5);
    pub const DAMAGE: Self = SkillTypeFlags(1 << 6);
    pub const CHAOS_INOCULATION: Self = SkillTypeFlags(1 << 7);

    pub fn contains(self, other: Self) -> bool {
        (self.0 & other.0) != 0
    }
}
```

```rust
use crate::build::{Build, types::ActiveSkill};
use crate::mod_db::{ModDb, types::{ModSource, ModFlags, KeywordFlags}};
use super::env::CalcEnv;

pub fn run(env: &mut CalcEnv, build: &Build) {
    // Resolve the active skill set and socket group
    let skill_set_idx = build.active_skill_set;
    let socket_group_idx = build.main_socket_group;

    let Some(skill_set) = build.skill_sets.get(skill_set_idx) else { return; };
    let Some(skill_group) = skill_set.skills.get(socket_group_idx) else { return; };

    // Find the active (non-support) gem at main_active_skill index
    let active_gem_idx = skill_group.main_active_skill;
    let active_gems: Vec<_> = skill_group.gems.iter().filter(|g| g.enabled && !g.is_support).collect();
    let Some(active_gem) = active_gems.get(active_gem_idx).or_else(|| active_gems.first()) else {
        return;
    };

    // Look up gem data to determine skill type
    // For now, classify by gem ID naming convention until full gem data is loaded.
    // Attacks: gems that are not in the known spell list.
    // This is a temporary heuristic — full classification uses gem.skill_types from GameData.
    let skill_id = active_gem.skill_id.clone();
    let known_spells: std::collections::HashSet<&str> = [
        "Fireball", "Frostbolt", "Arc", "Lightning Bolt", "Freezing Pulse",
        "Ball Lightning", "Storm Call", "Ice Nova", "Vaal Fireball",
        "Spark", "Incinerate", "Flameblast", "Scorching Ray",
    ].iter().copied().collect();

    let is_spell = known_spells.contains(skill_id.as_str());
    let is_attack = !is_spell;
    // Melee: not a bow or wand skill (simplification — full impl uses weapon slot)
    let known_ranged: std::collections::HashSet<&str> = [
        "Fireball", "Arc", "Frostbolt", "Ball Lightning",
    ].iter().copied().collect();
    let is_melee = is_attack && !known_ranged.contains(skill_id.as_str());

    // Build the skill mod db: start with a child of the player mod db
    // that adds the skill's attack/spell flag filtering.
    // Full implementation merges granted-effect mods from gem level tables.
    // For now, create a child mod db and set the IsMainSkill condition.
    let skill_mod_db = ModDb::new(); // TODO in Task 5: populate with gem level mods

    // Set conditions on the player mod db
    env.player.mod_db.set_condition("IsMainSkill", true);
    if is_attack {
        env.player.mod_db.set_condition("UsingAttack", true);
    }
    if is_spell {
        env.player.mod_db.set_condition("UsingSpell", true);
    }
    if is_melee {
        env.player.mod_db.set_condition("UsingMelee", true);
    }

    // Base cast time / attack speed for the skill.
    // Default values — replaced by gem data in Task 5.
    let cast_time = if is_spell { 0.7 } else { 0.0 };
    let attack_speed_base = if is_attack { 1.5 } else { 0.0 };

    env.player.main_skill = Some(ActiveSkill {
        skill_id,
        level: active_gem.level,
        skill_mod_db,
        is_attack,
        is_spell,
        is_melee,
        can_crit: true,
        base_crit_chance: if is_spell { 0.06 } else { 0.05 }, // default 5–6% base crit
        base_damage: std::collections::HashMap::new(), // populated in Task 5
        attack_speed_base,
        cast_time,
    });
}
```

- [ ] **Step 4: Implement `offence.rs` — hit damage core**

Replace the 3-line stub with the full implementation. Follow `CalcOffence.lua` lines 319–600 for the hit damage section. Key steps (each maps 1:1 to a Lua section):

```rust
use super::env::CalcEnv;
use crate::build::Build;
use crate::mod_db::types::{ModFlags, KeywordFlags, ModType};

pub fn run(env: &mut CalcEnv, build: &Build) {
    let Some(skill) = env.player.main_skill.as_ref() else { return; };
    
    // Skip disabled skills
    // (POB line 335: if skillFlags.disable then output.CombinedDPS = 0; return end)
    
    // --- Base damage per type ---
    // CalcOffence.lua lines 580–700: populate output[type+"MinBase"] and [type+"MaxBase"]
    // from skill base damage + weapon damage (attacks) or spell damage
    let dmg_types = ["Physical", "Lightning", "Cold", "Fire", "Chaos"];
    for dtype in &dmg_types {
        let base_min = skill.base_damage.get(*dtype).map(|(min, _)| *min).unwrap_or(0.0);
        let base_max = skill.base_damage.get(*dtype).map(|(_, max)| *max).unwrap_or(0.0);
        env.player.set_output(&format!("{}MinBase", dtype), base_min);
        env.player.set_output(&format!("{}MaxBase", dtype), base_max);
    }

    // --- Conversion table (Physical → Lightning → Cold → Fire → Chaos) ---
    // CalcOffence.lua lines 700–800: build the conversion percentage table.
    // For the initial implementation, no conversion (each type stays at itself).
    // Full conversion chain added when oracle tests require it.
    struct ConvTable {
        /// Fraction of this damage type that converts to each other type.
        conversion: [f64; 5], // indexed as [Phys, Light, Cold, Fire, Chaos]
        mult: f64,
    }
    // Default: 100% of each type stays as itself (no conversion)
    let conversion_tables: Vec<ConvTable> = dmg_types.iter().enumerate().map(|(i, _)| {
        let mut conv = ConvTable { conversion: [0.0; 5], mult: 1.0 };
        conv.conversion[i] = 1.0; // stays as itself
        conv
    }).collect();

    // --- Modifiers for each damage type ---
    // CalcOffence.lua lines 115–139: inc * more per type
    let skill_flags_attack = if skill.is_attack { ModFlags::ATTACK } else { ModFlags::NONE };
    let skill_flags_spell  = if skill.is_spell  { ModFlags::SPELL  } else { ModFlags::NONE };
    let combined_flags = ModFlags(skill_flags_attack.0 | skill_flags_spell.0);

    let mut total_min = 0.0_f64;
    let mut total_max = 0.0_f64;

    for (i, dtype) in dmg_types.iter().enumerate() {
        let base_min_key = format!("{}MinBase", dtype);
        let base_max_key = format!("{}MaxBase", dtype);
        let base_min = env.player.output.get(&base_min_key)
            .and_then(|v| if let super::env::OutputValue::Number(n) = v { Some(*n) } else { None })
            .unwrap_or(0.0);
        let base_max = env.player.output.get(&base_max_key)
            .and_then(|v| if let super::env::OutputValue::Number(n) = v { Some(*n) } else { None })
            .unwrap_or(0.0);

        if base_min == 0.0 && base_max == 0.0 { continue; }

        // Inc and more for this damage type + generic "Damage" modifier
        let inc_damage = env.player.mod_db.sum(ModType::Inc, "Damage", combined_flags, KeywordFlags::NONE);
        let inc_typed  = env.player.mod_db.sum(ModType::Inc, &format!("{}Damage", dtype), combined_flags, KeywordFlags::NONE);
        let inc_total  = inc_damage + inc_typed;
        let more_damage = env.player.mod_db.more("Damage", combined_flags, KeywordFlags::NONE);
        let more_typed  = env.player.mod_db.more(&format!("{}Damage", dtype), combined_flags, KeywordFlags::NONE);
        let more_total  = more_damage * more_typed;

        let min_val = (base_min * (1.0 + inc_total / 100.0) * more_total).round();
        let max_val = (base_max * (1.0 + inc_total / 100.0) * more_total).round();
        env.player.set_output(&format!("{}Min", dtype), min_val);
        env.player.set_output(&format!("{}Max", dtype), max_val);
        total_min += min_val;
        total_max += max_val;
    }

    // --- Crit chance and multiplier ---
    // CalcOffence.lua search for "CritChance"
    let base_crit = skill.base_crit_chance * 100.0; // in percent
    let inc_crit  = env.player.mod_db.sum(ModType::Inc, "CritChance", combined_flags, KeywordFlags::NONE);
    let more_crit = env.player.mod_db.more("CritChance", combined_flags, KeywordFlags::NONE);
    let crit_chance = (base_crit * (1.0 + inc_crit / 100.0) * more_crit).min(100.0).max(0.0);
    env.player.set_output("CritChance", crit_chance);

    let base_crit_multi = 150.0_f64; // 150% default
    let inc_crit_multi = env.player.mod_db.sum(ModType::Inc, "CritMultiplier", combined_flags, KeywordFlags::NONE);
    let crit_multi = (base_crit_multi + inc_crit_multi) / 100.0; // as a multiplier
    env.player.set_output("CritMultiplier", crit_multi);

    // --- Hit chance (attacks only) ---
    // CalcOffence.lua search for "HitChance"
    // hit_chance = clamp(accuracy / (accuracy + (evasion * factor)), 0.05, 1.0)
    // Simplified: default 100% for spells, computed for attacks
    let hit_chance = if skill.is_spell {
        1.0
    } else {
        let accuracy = env.player.mod_db.sum(ModType::Base, "Accuracy", ModFlags::NONE, KeywordFlags::NONE);
        let player_level = build.level as f64;
        // POB formula: chance = clamp(accuracy / (accuracy + enemy_evasion), 0.05, 1.0)
        // Enemy evasion from enemy level table — use simplified formula for now
        let enemy_evasion_approx = 15.0 + (build.level as f64).powf(2.0); // rough approximation
        if accuracy <= 0.0 {
            0.05
        } else {
            (accuracy / (accuracy + enemy_evasion_approx)).clamp(0.05, 1.0)
        }
    };
    env.player.set_output("HitChance", hit_chance * 100.0);

    // --- Average hit ---
    // CalcOffence.lua: averageHit = averageNonCrit * (1 - critRate) + averageCrit * critRate
    let avg_non_crit = (total_min + total_max) / 2.0;
    let avg_crit = avg_non_crit * crit_multi;
    let crit_rate = crit_chance / 100.0;
    let average_hit = avg_non_crit * (1.0 - crit_rate) + avg_crit * crit_rate;
    env.player.set_output("AverageHit", average_hit);

    let average_damage = average_hit * hit_chance;
    env.player.set_output("AverageDamage", average_damage);

    // --- Attack or cast speed ---
    let speed_mod = if skill.is_spell {
        env.player.output.get("CastSpeedMod")
            .and_then(|v| if let super::env::OutputValue::Number(n) = v { Some(*n) } else { None })
            .unwrap_or(1.0)
    } else {
        env.player.output.get("AttackSpeedMod")
            .and_then(|v| if let super::env::OutputValue::Number(n) = v { Some(*n) } else { None })
            .unwrap_or(1.0)
    };
    let uses_per_sec = if skill.is_spell {
        if skill.cast_time > 0.0 { speed_mod / skill.cast_time } else { 0.0 }
    } else {
        skill.attack_speed_base * speed_mod
    };
    env.player.set_output("Speed", uses_per_sec);

    // --- Total DPS ---
    // TotalDPS = AverageDamage * uses_per_second
    let total_dps = average_damage * uses_per_sec;
    env.player.set_output("TotalDPS", total_dps);
    env.player.set_output("CombinedDPS", total_dps); // same for simple builds

    // --- Breakdown for Physical ---
    if total_min > 0.0 || total_max > 0.0 {
        env.player.set_breakdown_lines("PhysicalDamage", vec![
            format!("{:.0} to {:.0} (base)", total_min, total_max),
            format!("Average hit: {:.0}", average_hit),
        ]);
    }
}
```
